# 楽天イーグルスホームページ

## 概要

このアプリケーションは、楽天イーグルスのホームページを参考にしてウェブページの基本的な機能を開発する目的で開発しました。

## 使用技術

- Spring Boot (Kotlin)
- Spring MVC / Thymeleaf
- Azure SQL DataBase
- Spring Data JPA
- HTML / CSS / JQuery
- AWS S3

## 開発内容

- 選手一覧、詳細ページ
- 選手情報CRUD
- 会員登録とログイン
- グッズ一覧
